# Empty 空状态

空状态时的占位提示。

## 组件说明

Empty 空状态组件用于数据为空时的占位提示。

## 属性

| 属性名      | 说明           | 类型   | 可选值                               | 默认值   |
| ----------- | -------------- | ------ | ------------------------------------ | -------- |
| description | 文本描述       | string | —                                    | 暂无数据 |
| image       | 图片地址       | string | —                                    | —        |
| image-size  | 图片大小(CSS值) | string | —                                    | —        |
| size        | 尺寸           | string | mini / small / medium / large / huge | medium   |

## 插槽

| 插槽名      | 说明           |
| ----------- | -------------- |
| default     | 自定义底部内容 |
| image       | 自定义图片     |
| description | 自定义描述文字 |
