# Row 行布局组件测试用例

## 组件信息

- **组件名称**: nv-row
- **组件路径**: `src/components/row`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 栅格系统的行容器组件

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `row-basic-render`
- **测试步骤**:
  1. 渲染一个 row 组件
  2. 在 row 中添加 col 组件
  3. 验证布局正常显示
- **预期结果**: row 正常渲染，col 正确排列

## 2. 属性测试

### 2.1 gutter 属性 - 间距

- **测试ID**: `row-gutter`
- **测试步骤**:
  1. 设置 gutter 属性
  2. 验证列之间的间距
- **预期结果**: 列之间显示正确的间距

### 2.2 justify 属性 - 水平对齐

- **测试ID**: `row-justify-{value}`
- **测试值**: start, end, center, space-around, space-between
- **测试步骤**:
  1. 设置不同的 justify 值
  2. 验证水平对齐方式
- **预期结果**: 列按指定方式水平对齐

### 2.3 align 属性 - 垂直对齐

- **测试ID**: `row-align-{value}`
- **测试值**: top, middle, bottom
- **测试步骤**:
  1. 设置不同的 align 值
  2. 验证垂直对齐方式
- **预期结果**: 列按指定方式垂直对齐

- **属性覆盖率**: 100%
