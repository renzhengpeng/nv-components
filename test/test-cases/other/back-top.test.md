# Backtop 回到顶部组件测试用例

## 组件信息

- **组件名称**: nv-backtop
- **测试优先级**: P2
- **组件描述**: 返回页面顶部的操作按钮

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `backtop-basic-render`
- **测试步骤**: 1. 渲染回到顶部按钮 2. 滚动页面 3. 验证按钮显示
- **预期结果**: 回到顶部按钮正常渲染

### 1.2 点击回到顶部

- **测试ID**: `backtop-click`
- **测试步骤**: 1. 滚动页面 2. 点击按钮 3. 验证回到顶部
- **预期结果**: 点击回到顶部正常
