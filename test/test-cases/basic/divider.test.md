# Divider 分割线组件测试用例

## 组件信息

- **组件名称**: nv-divider
- **组件路径**: `src/components/divider`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 区隔内容的分割线

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `divider-basic-render`
- **测试步骤**: 渲染基础分割线
- **预期结果**: 分割线正常显示

### 1.2 带文字分割线

- **测试ID**: `divider-with-text`
- **测试步骤**: 分割线中包含文字
- **预期结果**: 文字正常显示在分割线中

- **测试ID**: `divider-direction-{value}`

### 2.2 content-position 属性

- **测试ID**: `divider-position-{value}`
- **测试值**: left, center, right
- **测试步骤**: 设置文字位置

### 2.3 dashed 属性

- **测试ID**: `divider-dashed`
- **测试步骤**: 设置虚线样式
- **预期结果**: 显示虚线分割线
