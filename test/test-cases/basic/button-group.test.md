# Button Group 按钮组组件测试用例

## 组件信息

- **组件名称**: nv-button-group
- **组件路径**: `src/components/button-group`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 按钮组容器，用于组合多个按钮

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `button-group-basic-render`
- **测试步骤**:
  1. 渲染一个按钮组
  2. 在按钮组中添加多个按钮
  3. 验证按钮组正常显示
- **预期结果**: 按钮组正常渲染，内部按钮正确排列

### 1.2 按钮间距

- **测试ID**: `button-group-spacing`
- **测试步骤**:
  1. 验证按钮组内按钮之间的间距
  2. 验证按钮边框合并效果
- **预期结果**: 按钮之间无间隙，边框正确合并

## 2. 属性测试

### 2.1 size 属性 - 统一尺寸

- **测试ID**: `button-group-size-{value}`
- **测试值**: mini, small, medium, large, huge
- **测试步骤**:
  1. 设置按钮组的 size 属性
  2. 验证组内所有按钮尺寸一致
- **预期结果**: 组内所有按钮使用统一尺寸

### 2.2 vertical 属性 - 垂直排列

- **测试ID**: `button-group-vertical`
- **测试步骤**:
  1. 设置 vertical 属性
  2. 验证按钮垂直排列
- **预期结果**: 按钮垂直堆叠显示

## 3. 样式测试

### 3.1 水平布局

- **测试ID**: `button-group-horizontal`
- **测试步骤**:
  1. 默认水平布局
  2. 验证按钮水平排列
- **预期结果**: 按钮水平排列，边框正确合并

### 3.2 垂直布局

- **测试ID**: `button-group-vertical-layout`
- **测试步骤**:
  1. 设置垂直布局
  2. 验证按钮垂直排列
- **预期结果**: 按钮垂直排列，边框正确合并
