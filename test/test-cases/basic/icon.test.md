# Icon 图标组件测试用例

## 组件信息

- **组件名称**: nv-icon
- **组件路径**: `src/components/icon`
- **测试优先级**: P0 (高优先级)
- **组件描述**: 提供丰富的图标集，支持自定义颜色和尺寸

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `icon-basic-render`
- **测试步骤**: 渲染一个基础图标，验证图标正常显示
- **预期结果**: 图标正常渲染

### 1.2 图标名称

- **测试ID**: `icon-name-{iconName}`
- **测试步骤**: 设置不同的图标名称，验证对应图标显示
- **预期结果**: 显示正确的图标

## 2. 属性测试

### 2.1 name 属性

- **测试ID**: `icon-name-search`, `icon-name-edit`, `icon-name-delete` 等
- **测试步骤**: 测试常用图标名称
- **预期结果**: 每个图标正确显示
- **常用图标**: search, edit, delete, check, close, plus, minus, upload, download, refresh, setting, user, home, star, heart, warning, info, success, error

### 2.2 size 属性

- **测试ID**: `icon-size-{value}`
- **测试值**: 12, 14, 16, 18, 20, 24, 32, 48
- **测试步骤**: 设置不同尺寸
- **预期结果**: 图标尺寸正确

### 2.3 color 属性

- **测试ID**: `icon-color-{value}`
- **测试步骤**: 设置不同颜色值
- **预期结果**: 图标颜色正确

### 2.4 spin 属性（当前未实现）

- **测试ID**: `icon-spin`
- **测试步骤**: 设置 spin 属性（若组件后续支持）
- **预期结果**: 图标旋转动画
- **说明**: 当前 nv-icon 无 spin 属性，此用例为预留；实现后再启用

## 3. 事件测试

### 3.1 click 事件

- **测试ID**: `icon-click-event`
- **测试步骤**: 点击图标，验证事件触发
- **预期结果**: 点击事件正常触发

## 4. 样式测试

### 4.1 自定义颜色

- **测试ID**: `icon-custom-color`

### 4.2 自定义尺寸

- **测试ID**: `icon-custom-size`
- **测试步骤**: 使用 CSS 自定义图标尺寸
- **预期结果**: 尺寸可自定义
