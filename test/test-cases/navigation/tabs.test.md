# Tabs 标签页组件测试用例

## 组件信息

- **组件名称**: nv-tabs
- **组件路径**: `src/components/tabs`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 分隔内容上有关联但属于不同类别的数据集合

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `tabs-basic-render`
- **测试步骤**:
  1. 渲染 tabs 和多个 tab-pane
  2. 验证标签页正常显示
  3. 验证第一个标签页默认激活
- **预期结果**: 标签页正常渲染

### 1.2 标签页切换

- **测试ID**: `tabs-switch`
- **测试步骤**:
  1. 点击第二个标签
  2. 验证第二个标签页激活
  3. 验证第二个面板内容显示
- **预期结果**: 标签页切换正常

## 2. 属性测试

### 2.1 value 属性 (v-model)

- **测试ID**: `tabs-value`
- **测试步骤**:
  1. 设置 value 为某个标签页的 name
  2. 验证对应标签页激活
- **预期结果**: value 控制激活标签页

### 2.2 type 属性 - 类型

- **测试ID**: `tabs-type-{value}`
- **测试值**: card, border-card
- **测试步骤**:

  2. 验证标签页样式符合预期
- **预期结果**: 不同类型样式正确

  3. 点击关闭按钮
  4. 验证标签页被移除

### 2.4 addable 属性 - 可增加

- **测试ID**: `tabs-addable`
- **测试步骤**:
  1. 设置 addable 为 true

  3. 点击新增按钮

### 2.5 editable 属性 - 可编辑

- **测试ID**: `tabs-editable`
- **测试步骤**:
  1. 设置 editable 为 true
  2. 验证同时具有新增和关闭功能
- **预期结果**: 可编辑功能正常

### 2.6 tab-position 属性 - 位置

- **测试ID**: `tabs-position-{value}`
- **测试值**: top, right, bottom, left
- **测试步骤**:
  1. 设置不同的 tab-position 值
  2. 验证标签页位置正确
- **预期结果**: 不同位置正确显示
