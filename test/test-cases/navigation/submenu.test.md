# Submenu 子菜单组件测试用例

## 组件信息

- **组件名称**: nv-submenu
- **组件路径**: `src/components/submenu`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 导航菜单的子菜单组件，支持嵌套

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `submenu-basic-render`
- **测试步骤**:
  1. 在导航菜单中渲染子菜单
  2. 验证子菜单正常显示
- **预期结果**: 子菜单正常渲染

### 1.2 展开/收起

- **测试ID**: `submenu-toggle`
- **测试步骤**:
  1. 点击子菜单标题
  2. 验证子菜单展开
  3. 再次点击
  4. 验证子菜单收起
- **预期结果**: 子菜单正确展开和收起

### 1.3 嵌套子菜单

- **测试ID**: `submenu-nested`
- **测试步骤**:
  1. 创建多层嵌套子菜单
  2. 验证嵌套结构正确显示
- **预期结果**: 支持多层嵌套

## 2. 属性测试

### 2.1 index 属性 - 唯一标识

- **测试ID**: `submenu-index`
- **测试步骤**:
  1. 设置 index 属性
  2. 验证子菜单可以通过 index 识别
- **预期结果**: index 正确标识子菜单

### 2.2 title 属性 - 标题

- **测试ID**: `submenu-title`
- **测试步骤**:
  1. 设置 title 属性
  2. 验证标题正确显示
- **预期结果**: 标题正确显示
