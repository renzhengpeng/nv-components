# Nav Menu 导航菜单组件测试用例

## 组件信息

- **组件名称**: nv-nav-menu
- **组件路径**: `src/components/nav-menu`
- **测试优先级**: P0 (高优先级)
- **组件描述**: 导航菜单容器组件，用于组织菜单项和子菜单

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `nav-menu-basic-render`
- **测试步骤**:
  1. 渲染导航菜单
  2. 添加菜单项
  3. 验证菜单正常显示
- **预期结果**: 导航菜单正常渲染

### 1.2 菜单项选择

- **测试ID**: `nav-menu-item-select`
- **测试步骤**:
  1. 点击菜单项
  2. 验证菜单项被选中
  3. 验证触发 select 事件
- **预期结果**: 菜单项正确选中，事件正常触发

## 2. 属性测试

### 2.1 mode 属性 - 菜单模式

- **测试ID**: `nav-menu-mode-{value}`
- **测试值**: horizontal, vertical
- **测试步骤**:
  1. 设置不同的 mode 值
  2. 验证菜单布局方向
- **预期结果**: 菜单按指定方向显示

### 2.2 collapse 属性 - 折叠状态

- **测试ID**: `nav-menu-collapse`
- **测试步骤**:
  1. 设置 collapse 属性
  2. 验证菜单折叠显示
- **预期结果**: 菜单正确折叠

### 2.3 default-active 属性 - 默认激活

- **测试ID**: `nav-menu-default-active`
- **测试步骤**:
  1. 设置 default-active 属性
  2. 验证对应菜单项被激活
- **预期结果**: 指定菜单项默认激活

## 3. 事件测试

### 3.1 select 事件

- **测试ID**: `nav-menu-select-event`
- **测试步骤**:
  1. 监听 select 事件
  2. 点击菜单项
  3. 验证事件触发并传递正确参数
- **预期结果**: select 事件正常触发
