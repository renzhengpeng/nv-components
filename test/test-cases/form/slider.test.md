# Slider 滑块组件测试用例

## 组件信息

- **组件名称**: nv-slider
- **测试优先级**: P1

- **测试ID**: `slider-basic-render`
- **测试步骤**: 1. 渲染滑块 2. 验证滑块正常显示
- **预期结果**: 滑块正常渲染
