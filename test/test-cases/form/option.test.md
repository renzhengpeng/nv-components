# Option 选项组件测试用例

## 组件信息

- **组件名称**: nv-option
- **组件路径**: `src/components/option`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 下拉选择器的选项组件

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `option-basic-render`
- **测试步骤**:
  1. 在 select 中渲染选项
  2. 验证选项正常显示
- **预期结果**: 选项正常渲染

### 1.2 选择功能

- **测试ID**: `option-select`
- **测试步骤**:
  1. 点击选项
  2. 验证选项被选中
- **预期结果**: 选项正确响应选择

## 2. 属性测试

### 2.1 value 属性 - 选项值

- **测试ID**: `option-value`
- **测试步骤**:
  1. 设置 value 属性
  2. 验证选中时返回正确的值
- **预期结果**: value 正确传递

### 2.2 label 属性 - 显示文本

- **测试ID**: `option-label`
- **测试步骤**:
  1. 设置 label 属性
  2. 验证显示文本正确
- **预期结果**: label 正确显示

### 2.3 disabled 属性 - 禁用状态

- **测试ID**: `option-disabled`
- **测试步骤**:
  1. 设置 disabled 属性
  2. 验证选项不可选择
- **预期结果**: 禁用的选项不响应点击

## 测试覆盖率目标

- **功能覆盖率**: 100%
- **属性覆盖率**: 100%
