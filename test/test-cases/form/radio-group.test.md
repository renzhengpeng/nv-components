# Radio Group 单选框组组件测试用例

## 组件信息

- **组件名称**: nv-radio-group
- **组件路径**: `src/components/radio-group`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 单选框组容器，用于管理多个单选框

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `radio-group-basic-render`
- **测试步骤**:
  1. 渲染单选框组
  2. 在组中添加多个单选框
  3. 验证组正常显示
- **预期结果**: 单选框组正常渲染

### 1.2 值绑定

- **测试ID**: `radio-group-value`
- **测试步骤**:
  1. 设置 value 属性
  2. 验证对应的单选框被选中
- **预期结果**: 值正确绑定到单选框

### 1.3 互斥选择

- **测试ID**: `radio-group-exclusive`
- **测试步骤**:
  1. 选中一个单选框
  2. 选中另一个单选框
  3. 验证只有一个被选中
- **预期结果**: 组内单选框互斥

## 2. 属性测试

### 2.1 value 属性 - 选中值

- **测试ID**: `radio-group-value-binding`
- **测试步骤**:
  1. 设置 value 值
  2. 验证对应单选框选中状态
- **预期结果**: 单选框选中状态与 value 一致

### 2.2 disabled 属性 - 禁用状态

- **测试ID**: `radio-group-disabled`
- **测试步骤**:
  1. 设置 disabled 属性
  2. 验证组内所有单选框被禁用
- **预期结果**: 组内所有单选框不可操作

## 3. 事件测试

### 3.1 change 事件

- **测试ID**: `radio-group-change-event`
- **测试步骤**:
  1. 监听 change 事件
  2. 选中单选框
  3. 验证事件触发并传递正确值
- **预期结果**: change 事件正常触发

- **属性覆盖率**: 100%
