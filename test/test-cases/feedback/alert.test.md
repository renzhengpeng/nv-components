# Alert 警告提示组件测试用例

## 组件信息

- **组件名称**: nv-alert
- **测试优先级**: P1
- **组件描述**: 用于页面中展示重要的提示信息

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `alert-basic-render`
- **测试步骤**: 1. 渲染基础警告提示 2. 验证提示正常显示
- **预期结果**: 警告提示正常渲染

### 1.2 可关闭

- **测试ID**: `alert-closable`
- **测试步骤**: 1. 设置 closable 2. 点击关闭按钮 3. 验证提示关闭
- **预期结果**: 可关闭功能正常
