# Collapse Item 折叠面板项组件测试用例

## 组件信息

- **组件名称**: nv-collapse-item
- **组件路径**: `src/components/collapse-item`
- **测试优先级**: P1 (中优先级)
- **组件描述**: 折叠面板的单个项组件

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `collapse-item-basic-render`
- **测试步骤**:
  1. 在折叠面板中渲染项
  2. 验证项正常显示
- **预期结果**: 折叠面板项正常渲染

### 1.2 展开/收起

- **测试ID**: `collapse-item-toggle`
- **测试步骤**:
  1. 点击项标题
  2. 验证内容展开
  3. 再次点击
  4. 验证内容收起
- **预期结果**: 项正确展开和收起

## 2. 属性测试

### 2.1 name 属性 - 唯一标识

- **测试ID**: `collapse-item-name`
- **测试步骤**:
  1. 设置 name 属性
  2. 验证可以通过 name 控制展开状态
- **预期结果**: name 正确标识项

### 2.2 title 属性 - 标题

- **测试ID**: `collapse-item-title`
- **测试步骤**:
  1. 设置 title 属性
  2. 验证标题正确显示
- **预期结果**: 标题正确显示

### 2.3 disabled 属性 - 禁用状态

- **测试ID**: `collapse-item-disabled`
- **测试步骤**:
  1. 设置 disabled 属性
  2. 验证项不可展开
- **预期结果**: 禁用的项不响应点击
