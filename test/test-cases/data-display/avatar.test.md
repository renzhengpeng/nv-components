# Avatar 头像组件测试用例

## 组件信息

- **组件名称**: nv-avatar
- **测试优先级**: P1
- **组件描述**: 用图标、图片或者字符的形式展示用户或事物信息

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `avatar-basic-render`
- **测试步骤**: 1. 渲染基础头像 2. 验证头像正常显示
- **预期结果**: 头像正常渲染

### 1.2 图片头像

- **测试ID**: `avatar-image`
- **测试步骤**: 1. 设置 src 属性 2. 验证图片显示
- **预期结果**: 图片头像正常显示

### 1.3 文字头像

- **测试ID**: `avatar-text`
- **测试步骤**: 1. 在默认插槽添加文字 2. 验证文字显示
- **预期结果**: 文字头像正常显示

### 1.4 图标头像

- **测试ID**: `avatar-icon`
- **测试步骤**: 1. 设置 icon 属性 2. 验证图标显示
- **预期结果**: 图标头像正常显示

## 2. 属性测试

### 2.1 size 属性

- **测试ID**: `avatar-size-{value}`
- **测试值**: large, medium, small, number
- **测试步骤**: 1. 设置不同 size 值 2. 验证尺寸正确
- **预期结果**: 不同尺寸正确显示

### 2.2 shape 属性

- **测试ID**: `avatar-shape-{value}`
- **测试值**: circle, square
- **测试步骤**: 1. 设置不同 shape 值 2. 验证形状正确
- **预期结果**: 不同形状正确显示

### 2.3 src 属性

- **测试ID**: `avatar-src`
- **测试步骤**: 1. 设置 src 2. 验证图片加载
- **预期结果**: 图片正确加载

### 2.4 alt 属性

- **测试ID**: `avatar-alt`
- **测试步骤**: 1. 设置 alt 2. 验证 alt 文本
- **预期结果**: alt 属性正确

## 3. 事件测试

### 3.1 error 事件

- **测试ID**: `avatar-event-error`
- **测试步骤**: 1. 设置无效 src 2. 监听 error 事件 3. 验证事件触发
- **预期结果**: 图片加载失败触发 error 事件

## 测试覆盖率目标

- **功能覆盖率**: ≥90%
- **属性覆盖率**: ≥90%
