# Collapse 折叠面板组件测试用例

## 组件信息

- **组件名称**: nv-collapse
- **测试优先级**: P2
- **组件描述**: 通过折叠面板收纳内容区域

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `collapse-basic-render`
- **测试步骤**: 1. 渲染折叠面板 2. 验证面板正常显示
- **预期结果**: 折叠面板正常渲染

### 1.2 展开/折叠

- **测试ID**: `collapse-toggle`
- **测试步骤**: 1. 点击面板标题 2. 验证面板展开/折叠
- **预期结果**: 展开折叠正常

## 2. 属性测试

### 2.1 value 属性 (v-model)

- **测试ID**: `collapse-value`
- **测试步骤**: 1. 设置 value 2. 验证对应面板展开
- **预期结果**: value 控制展开面板

### 2.2 accordion 属性

- **测试ID**: `collapse-accordion`
- **测试步骤**: 1. 设置 accordion 2. 展开多个面板 3. 验证只保留一个
- **预期结果**: 手风琴模式正常

## 3. 事件测试

### 3.1 change 事件

- **测试ID**: `collapse-event-change`
- **测试步骤**: 1. 监听事件 2. 展开/折叠面板 3. 验证事件触发
- **预期结果**: change 事件正常

## 测试覆盖率目标

- **功能覆盖率**: ≥90%
- **属性覆盖率**: ≥90%
