# Badge 徽标组件测试用例

## 组件信息

- **组件名称**: nv-badge
- **测试优先级**: P1
- **组件描述**: 出现在按钮、图标旁的数字或状态标记

## 1. 功能测试

### 1.1 基础渲染

- **测试ID**: `badge-basic-render`
- **测试步骤**: 1. 渲染基础徽标 2. 验证徽标正常显示
- **预期结果**: 徽标正常渲染

### 1.2 数字徽标

- **测试ID**: `badge-value`
- **测试步骤**: 1. 设置 value 属性 2. 验证数字显示
- **预期结果**: 数字徽标正常显示

### 1.3 小圆点

- **测试ID**: `badge-dot`
- **测试步骤**: 1. 设置 is-dot 属性 2. 验证显示小圆点
- **预期结果**: 小圆点正常显示

## 2. 属性测试

### 2.1 value 属性

- **测试ID**: `badge-value-prop`
- **测试步骤**: 1. 设置 value 2. 验证数字显示
- **预期结果**: value 正确显示

### 2.2 max 属性

- **测试ID**: `badge-max`
- **测试步骤**: 1. 设置 max="99" 2. 设置 value="100" 3. 验证显示"99+"
- **预期结果**: 超过 max 显示"max+"

### 2.3 is-dot 属性

- **测试ID**: `badge-is-dot`
- **测试步骤**: 1. 设置 is-dot 2. 验证显示小圆点
- **预期结果**: 小圆点模式正常

### 2.4 hidden 属性

- **测试ID**: `badge-hidden`
- **测试步骤**: 1. 设置 hidden 2. 验证徽标隐藏
- **预期结果**: 徽标可以隐藏

### 2.5 type 属性

- **测试ID**: `badge-type-{value}`
- **测试值**: primary, success, warning, danger, info
- **测试步骤**: 1. 设置不同 type 2. 验证颜色正确
- **预期结果**: 不同类型颜色正确
